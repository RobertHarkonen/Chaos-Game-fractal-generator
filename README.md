## Komentoriviohjeet

- Testaus: ´mvn test´
- Testikattavuusraportin generointi: ´mvn test jacoco:report´
- .jar -tiedoston generointi: ´mvn package´
- Checkstyle: ´mvn checkstyle:checkstyle´
- Javadocin generointi: ´mvn javadoc:javadoc´
